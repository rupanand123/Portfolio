# Rupanand Palakurthi | AI & Full Stack Developer Portfolio

An editorial-style, high-fidelity portfolio website built with React, Tailwind CSS, and Framer Motion. This site showcases the professional profile, projects, and technical competencies of Rupanand Palakurthi.

## ✨ Features

- **Editorial / Magazine Aesthetic**: A sophisticated dark-themed design with bold typography and high-contrast layouts.
- **Interactive Project Showcase**: Expandable project cards that reveal problem statements, technical solutions, and technology stacks.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile viewing.
- **Dynamic Content**: Data-driven architecture using a central content repository.
- **PDF Resume Export**: Integrated "Download CV" functionality via optimized print styles.
- **Fluid Animations**: Smooth transitions and hover effects powered by `motion/react`.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Type Safety**: [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite 6](https://vitejs.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/App.tsx`: Main application shell and layout.
- `src/types.ts`: TypeScript interfaces and the central `resumeData` object.
- `src/index.css`: Global styles and Tailwind theme configurations.
- `src/components/`: Reusable UI components (if applicable).
